What is a stream?
A stream is basicaly a web server, is defined by a IP and a port. You can easily create a stream by using the pystream client, doing this, you can share files between computers (on LAN and Internet). Also, you can protect your stream with a password.
Where can i get the pystream client?
Here :P
Can i develop my own client?
Off course! To create a stream you only need to send a POST request to http://www.pystream.com/new with this data:
version:API version.
ip:Your public IP.
port:Your public port.
lan_ip:Your IP from LAN (this is optional).
