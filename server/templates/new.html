What is pystream?
Pystream is a web service for sharing files between computers. You can easily create a stream by using the pystream client

There are two types of streams:
  • Public streams: avaliable for everyone, on searches and random. The easiest way to share files with everyone.
  • Private streams: hidden from searches (except from you LAN) and random, perfect for local sharing or LAN Parties.
    You can also use a password to protect you private stream.
LAN Parties:
Download the pystream client, create the streams and enjoy.
All streams (public or private) from your LAN will be shown on the main page, and will be avaliable for searches (from you LAN).

Can I develop my own client?
Off course! To create a stream you only need to send a POST request to http://www.pystream.com/new with this data:
version:API version ({{ pystream_version }}).
ip:Your public IP.
port:Your public port.
